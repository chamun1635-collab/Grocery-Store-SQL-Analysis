## Objectives
- Practice SQL queries (joins, aggregations, subqueries)
- Extract insights from grocery store data
- Provide actionable business recommendations

## Dataset
The project simulates a grocery store with tables such as:
- Customers
- Products
- Orders
- Suppliers
- Employees
- Categories
- Order details

## Analysis Questions
- Customer insights: top customers, purchase patterns
- Product performance: best-selling products, revenue per product
- Sales trends: daily, monthly, weekday/weekend patterns
- Supplier and employee contributions

## Tech Stack
- SQL (MySQL / PostgreSQL / SQLite)
- PowerPoint for presenting insights

## Project Files
- Queries.sql → Contains all SQL queries for analysis
- Report.pptx → Presentation with findings and visualizations
